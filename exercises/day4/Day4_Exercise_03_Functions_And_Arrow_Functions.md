# Day 4 Exercise 03 - Write Functions and Arrow Functions for Student Data

## What You Just Learned

The trainer showed:

```text
normal function
arrow function
short arrow function
```

using a course object.

Now you will write functions for a student object.

## Your Task

Create a file:

```text
student-functions.js
```

Create one student object with:

```text
studentId
studentName
email
status
```

## Coding Requirements

Write these functions:

### 1. Normal Function

```javascript
function formatStudent(student) {
  // return a formatted string
}
```

It should return something like:

```text
S001 - Aina Rahman (Active)
```

### 2. Arrow Function

```javascript
const getStudentEmail = (student) => {
  // return student email
};
```

### 3. Short Arrow Function

```javascript
const getStudentStatus = (student) => student.status;
```

## Required Output

Print:

```text
S001 - Aina Rahman (Active)
aina@example.com
Active
```

## Important Rule

Do not only copy the trainer's `formatCourse` function. You must change property names correctly for `Student`.
