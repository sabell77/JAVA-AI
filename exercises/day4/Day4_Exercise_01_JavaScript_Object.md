# D4 Exercise 01 - Create a JavaScript Student Object

## What You Just Learned

The trainer showed how a JavaScript object can represent one course.

In Java, you needed a class such as `Course.java`.

In JavaScript, simple data can be represented using an object literal.

## Your Task

Create a file:

```text
student-object.js
```

Write a JavaScript object called `student`.

The object must have:

```text
studentId
studentName
email
status
```

Example values:

```text
S001
Ignacio de Paul
ignacio@example.com
Active
```

## Coding Requirements

1. Create the object using `{}`.
2. Print the whole object.
3. Print each property one by one.
4. Use dot notation at least two times.
5. Use bracket notation at least one time.

## Example Output

```text
=== Student Object ===
{ studentId: 'S001', studentName: 'Ignacio de Paul', email: 'ignacio@example.com', status: 'Active' }

Student ID: S001
Name: Ignacio de Paul
Email: ignacio@example.com
Status: Active
```

## Important Rule

Do not create a `Course` object. The trainer already showed `Course`. You must apply the same idea to `Student`.
