# Day 4 Exercise 02 - Store Instructors in an Array and Loop Through Them

## What You Just Learned

The trainer showed how to store many courses inside a JavaScript array.

Now you will apply the same idea to instructors.

## Your Task

Create a file:

```text
instructor-array.js
```

Create an array called:

```javascript
const instructors = [];
```

The array must contain at least 4 instructor objects.

Each instructor must have:

```text
instructorId
instructorName
expertise
```

## Coding Requirements

1. Create at least 4 instructor objects inside an array.
2. Use a `for...of` loop to print all instructors.
3. Print each instructor in a readable format.
4. Print the total number of instructors using `.length`.

## Example Output

```text
=== Instructor List ===
I001 - Ignacio de Paul - Java and Spring Boot
I002 - Roberto Tan - React Development
I003 - Juan Carlos Lee - MongoDB
I004 - Carlos Kim - Testing

Total instructors: 4
```

## Important Rule

Do not use a normal Java-style array such as fixed-size arrays. In JavaScript, use a normal array with `[]`.
